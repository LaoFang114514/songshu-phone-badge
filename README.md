一款简单易用的手机端兽牌制作工具，帮你把闲置旧手机变成专属兽牌展示牌，适合需要兽牌但经费不足的兽圈小伙伴。

---

## ✨ 功能特性
### 🖼️ 兽牌管理
- **图片 / 视频双模式** — 支持导入图片或视频作为兽牌，自由切换展示
- **新建与编辑** — 内置裁剪工具，可选择图片区域裁剪为兽牌
- **列表管理** — 重命名、上移、下移、删除，轻松管理多张兽牌
- **选中高亮** — 当前使用的兽牌带有主题色边框，一目了然

### 🎬 全屏播放
- **沉浸式体验** — 全屏展示兽牌，自动隐藏系统栏
- **手势交互** — 长按屏幕旋转兽牌（180° 翻转），双指捏合退出播放
- **二维码展示** — 上划屏幕弹出二维码，支持自定义二维码图片

### ⚙️ 实用设置
- **反向显示** — 启动时自动翻转 180°，适配倒置摆放
- **常亮显示** — 播放时屏幕保持常亮，不自动熄灭
- **电量显示** — 全屏时顶部显示当前电量百分比
- **锁定方向** — 禁止长按旋转，固定兽牌朝向
- **防烧屏** — 每隔 5 分钟轻微移动画面，防止 OLED 残影

### 💾 数据备份
- **本地导入/导出** — 配置和图片打包为 ZIP 文件，方便迁移
- **WebDAV 云备份** — 支持上传到 WebDAV 服务器或从服务器恢复

## 📱 系统要求

- **最低版本**：Android 6.0 
- **推荐版本**：Android 8.0 及以上
- **架构支持**：armeabi-v7a / arm64-v8a

> 只在 32 位 Android 6.0 设备上测试通过，其他手机不确定能不能运行。
